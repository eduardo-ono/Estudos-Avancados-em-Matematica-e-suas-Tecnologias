## Múltiplas Funções

$$
f(x) = x + 1, \quad x \in [-5, 8] \\[10pt]
g(x) = x^2 - 5x + 6, \quad x \in [-5, 8]
$$

Comandos no GeoGebra

```latex
f(x) = x + 1, -5 <= x <= 8
g(x) = x^2 - 5x + 6, -5 <= x <= 8
```

ou

```latex
f(x) = Function(x + 1, -5, 10)
g(x) = Function(x^2 -5x + 6, -5, 10)
```
