## Intervalo de Funções

$$
M(t) = C \cdot (1 + j)^t, \quad t \in [0, 120]
$$

Comando no GeoGebra:

```latex
c = 10000
j = 0.006
m(t) = c (1 + j)^t, 0 <= t <= 120
```

ou

```latex
c = 10000
j = 0.006
Se(0 <= t <= 120, m(t) = c (1 + j)^t)
```
